# Projet_bibliographique_M1

Ce dépôt présente mon projet bibliographique réalisé dans le cadre du M1 Bioinformatique à l'université de Montpellier. Ce projet est consacré à l’analyse de l’ADN environnemental (ADNe) et à l’apport des technologies de séquençage longues lectures. Ce travail s’intéresse à la manière dont les approches *long read* peuvent améliorer la détection, l’identification et la caractérisation de la biodiversité, en comparaison avec les technologies à lectures courtes. Il aborde également les principaux traitements bioinformatiques associés, ainsi qu’une comparaison de pipelines comme FROGS et SAMBA.
